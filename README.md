# Sistema OS Tivoli

Sistema de Ordens de Serviço para BS TEC Tivoli

## Tecnologias

- React 18
- Vite
- Tailwind CSS
- Supabase
- React Router DOM
- Framer Motion
- Radix UI

## Instalação

```bash
npm install
```

## Configuração

1. Copie o arquivo `.env.example` para `.env`
2. Configure as variáveis de ambiente do Supabase

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```
